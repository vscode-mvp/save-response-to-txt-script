
# 💫 Сохранение ответов в файл txt

Скрипт позволяющий совершать запросы GET и сохранить ответы в файл jsonResponses.txt (создается автоматически), при совершении нескольких запросов поочередно, они будут сохранятся каждый раз на новой строке.




## 🛠️ Как использоавть

```javascript
apiFetchAndWrite() // В АРГУМЕНТАХ ВАШ URL ПО КОТОРОМУ НЕОБХОДИМО ПОЛУЧИТЬ ОТВЕТ
```
При вызове главной функции *apiFetchAndWrite()* в аргументах необходимо передать URL в внутри "", по которому необходимо получить ответ и записать в txt

## 📍 Запуск MacOS / Linux

1️⃣ Когда скрипт подготовлен необходимо скачать репозиторий
(по умолчанию папка с проектом сохранится в домашнюю папку)

```bash
  git clone https://github.com/vscode-mvp/save-response-to-txt-script
```
2️⃣ Перейдите в папку с проектом 

```bash
  cd save-response-to-txt-script
```
3️⃣ Установите зависимости

```bash
  npm install
```
4️⃣ Запустите скрипт через интерпритатор [Node.js](https://nodejs.org/en/download)

```bash
  node save-response-in-txt.js
```
## 📍 Запуск на Win10+

1️⃣ Скачайте репозиторий в Zip, распакуйте его и перейдите в папку

2️⃣ В адресной стрке напишите cmd и нажмите Enter

3️⃣ В терминале выполните поочередно команды 

Установка зависимостей
```bash
  npm install
```

Запуск главного исполняемого файла
```bash
  node save-response-in-txt.js
```
